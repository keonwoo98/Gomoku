## Features

- 19x19 board with Pygame UI
- AI using Alpha-Beta Pruning with Iterative Deepening
- Ninuki-renju capture rules (W-B-B-W pattern)
- Double-three prohibition
- Multiple game modes: PvP, PvE, EvE

## Requirements

- Python 3.10+
- Pygame 2.x

## Usage

```bash
make run
```

## Controls

- Click: Place stone
- D: Toggle debug panel
- V: Show valid moves
- ESC: Quit
